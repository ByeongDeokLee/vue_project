Vue3에 대해
setUp함수

- Composition API에서 컴포넌트의 로직을 정의하는 함수
- 메서드 정의할때 setup함수에서는 내부에 함수 선언가능
- 모든 로직이 setup() 안에 모임(data, methods, computed 등)
